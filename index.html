⚙️ Google Sheets Setup Guide
Follow these steps once — takes about 10 minutes. No payment needed. 100% free.
1

Create a Google Sheet

Go to sheets.google.com → Create a new blank spreadsheet → Name it AMICS Enrollments. You'll need 3 tabs: Students, Tutors, Messages.

2

Open Apps Script

Inside your Google Sheet, click Extensions → Apps Script. Delete any existing code and paste the script below:

function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet(); var data = JSON.parse(e.postData.contents); var tab = sheet.getSheetByName(data.sheet) || sheet.insertSheet(data.sheet); if (tab.getLastRow() === 0) { tab.appendRow(Object.keys(data.row)); } tab.appendRow(Object.values(data.row)); return ContentService .createTextOutput(JSON.stringify({status:'ok'})) .setMimeType(ContentService.MimeType.JSON); } function doGet() { return ContentService .createTextOutput('AMICS API is running.') .setMimeType(ContentService.MimeType.TEXT); }
3

Deploy as Web App

Click Deploy → New Deployment → Web App. Set Execute as: Me and Who has access: Anyone. Click Deploy → Copy the Web App URL.

4

Paste URL into this file

Open this HTML file in any text editor (Notepad, VS Code). Find this line near the top of the <script> section:

const SHEETS_URL = 'https://script.google.com/macros/s/AKfycbyI8ZFslnu9r8CZ-23qR47JzqAJ388GSrh_rrmC-q8xzbKgxvCyXTOjaD2dyxTBGYIEWA/exec';

Replace https://script.google.com/macros/s/AKfycbyI8ZFslnu9r8CZ-23qR47JzqAJ388GSrh_rrmC-q8xzbKgxvCyXTOjaD2dyxTBGYIEWA/exec with the URL you copied. Save the file. Done! ✅

5

Host the file for free

Go to netlify.com → drag and drop this HTML file → your site is live in seconds at a free URL. Buy a domain from Namecheap.com (~$10/yr) and connect it in Netlify settings.
